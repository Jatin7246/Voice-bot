#Voice Bot (OpenAI + Speech Recognition)

A conversational, voice-activated chatbot built using Streamlit, OpenAI API, and SpeechRecognition. It simulates responses in the personality and tone of Jatin Umashankar, an aspiring AI/ML Engineer and Data Scientist. The bot can answer introspective questions and interact naturally using voice input.

## Features

-  Real-time voice input using microphone
-  Natural responses powered by OpenAI GPT model
-  Personality-driven prompts
-  Instant replies via Streamlit interface
-  Secrets handled via `secrets.toml` (not `.env`)
